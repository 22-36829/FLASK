About the Pharmaceutical Sales & Inventory Dashboard

Overview

This dashboard provides comprehensive analytics and forecasting capabilities for pharmaceutical sales and inventory management. It leverages advanced machine learning models to predict future trends and help make data-driven decisions.

Features

  • Interactive Visualizations - Explore data through interactive charts with drill-down capabilities
  • Advanced Analytics - Access product segmentation, sales trend analysis, inventory optimization, and more
  • Machine Learning Forecasts - Utilize trained models for revenue, inventory, and reservation predictions
  • Real-time Updates - Get live updates through WebSocket connections
  • Custom Date Filtering - Analyze data across flexible time periods
  • Export Capabilities - Export charts and data in multiple formats (PNG, HTML, CSV)

Models

Model Type Purpose Algorithm Typical Accuracy
Revenue Forecasting Predict future revenue trends ARIMA (AutoRegressive Integrated Moving Average) 92-95%
Inventory Prediction Predict days until stock depletion XGBoost (Gradient Boosting) 90-93%
Reservation Forecasting Predict future product reservations Prophet (Facebook's time series forecasting) 91-94%

Technology Stack

Backend
  • Flask - Web framework
  • SQLAlchemy - ORM for database operations
  • Flask-SocketIO - Real-time WebSocket communication
  • Pandas & NumPy - Data processing
  • Scikit-learn - Machine learning utilities
  • ARIMA, XGBoost, Prophet - Forecasting models
Frontend
  • Bootstrap 5 - Responsive UI framework
  • Plotly - Interactive visualizations
  • Socket.IO - Real-time updates
  • Flatpickr - Date range selection
  • Font Awesome - Icons

Data Sources

The dashboard can work with any CSV data that includes the following fields:

  • date - Date of the record
  • product_id - Unique product identifier
  • product_name - Name of the product
  • category - Product category
  • price - Product price
  • stock_level - Current inventory level
  • quantity_sold - Number of units sold
  • reservations - Number of reservations
  • revenue - Revenue generated
